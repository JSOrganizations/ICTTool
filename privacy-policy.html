Local Device Storage

লোকাল ডিভাইস স্টোরেজ

The following data may be stored on your device for core app functionality and the notification system:

অ্যাপের মূল কার্যকারিতা এবং নোটিফিকেশন সিস্টেমের জন্য নিচের ডেটাগুলো আপনার ডিভাইসে সংরক্ষিত হতে পারে:

  • App Preferences — Theme, language, time format, and feature settings.
  • Calculation History — Previous conversion and arithmetic results.
  • HTML Runner History — HTML snippets entered and saved by you.
  • Study Notes — Notes you create in the Notes section.
  • Reading Progress — Local state related to textbook viewing.
  • Last Used Time — Used locally to calculate idle time for 'Milestone Notifications'.
  • Last Notification Time — Saved locally to ensure you are not bothered with repeated notifications on the same day.
  • Version & Package Data — The app reads its current version code and package name locally to compare with our server for 'Update Available' alerts. This is not stored as personal identity data.
  • অ্যাপের পছন্দসমূহ — থিম, ভাষা, সময় বিন্যাস এবং বিভিন্ন সেটিংস।
  • হিসাবের ইতিহাস — আগের রূপান্তর এবং গাণিতিক ফলাফল।
  • HTML রানার হিস্ট্রি — আপনার লেখা এবং সেভ করা HTML কোড।
  • স্টাডি নোটস — নোটস সেকশনে তৈরি করা আপনার নিজস্ব নোট।
  • পড়ার অগ্রগতি — পাঠ্যপুস্তক দেখার লোকাল অবস্থা।
  • সর্বশেষ ব্যবহারের সময় — আপনি কতদিন ধরে অ্যাপটি ব্যবহার করছেন না তা হিসেব করে 'মাইলস্টোন নোটিফিকেশন' পাঠানোর জন্য এটি ব্যবহৃত হয়।
  • সর্বশেষ নোটিফিকেশনের সময় — একই দিনে বারবার নোটিফিকেশন পাঠিয়ে আপনাকে বিরক্ত না করার জন্য এটি ডিভাইসে সেভ থাকে।
  • ভার্সন ও প্যাকেজ ডেটা — আপডেট চেক করার জন্য আপনার ডিভাইসে ইনস্টল করা অ্যাপের ভার্সন কোড এবং প্যাকেজ নাম রিড করা হয়। এটি ব্যক্তিগত তথ্য হিসেবে সংরক্ষিত হয় না।
Offline & SecureThese notification and version data points are strictly non-personal, completely offline, and are never transmitted outside your device. You can delete this data anytime by clearing app data.
অফলাইন এবং সুরক্ষিতএই নোটিফিকেশন এবং ভার্সন ডেটাগুলো সম্পূর্ণ অফলাইন এবং আপনার ডিভাইসের বাইরে কোথাও পাঠানো হয় না। অ্যাপ ডেটা ক্লিয়ার করে আপনি যেকোনো সময় এটি মুছে ফেলতে পারেন।

How your device stores app data

Data is stored in app-local storage (including SharedPreferences). Some local preferences are protected using encrypted storage where supported.

ডেটা অ্যাপের লোকাল স্টোরেজে সেভ করা হয়। কিছু লোকাল প্রেফারেন্স এনক্রিপ্টেড স্টোরেজ দিয়ে সুরক্ষিত থাকে।

  • Settings and preferences are saved in app-scoped storage.
  • User-created notes/history remain on your device.
  • No dedicated developer-operated cloud database is used for this data.
  • সেটিংস এবং প্রেফারেন্স অ্যাপ-স্কোপড স্টোরেজে সেভ হয়।

  • এই ডেটার জন্য ডেভেলপার-পরিচালিত কোনো ক্লাউড ডেটাবেস ব্যবহার করা হয় না।

পারমিশনের ব্যাখ্যা
Why each permission is requested
কেন পারমিশনগুলো চাওয়া হয়

ICT Tool requests only these permissions:

ICT Tool শুধুমাত্র এই পারমিশনগুলো চায়:

  • Internet Access — Used to access online resources and sync remote notification content securely.
  • Network State Access — Used to check connectivity before online requests.
  • Notification Permission (POST_NOTIFICATIONS) — For Android 13+, used solely to send you study reminders and app feature updates.
  • Foreground Service/Work Manager — Utilizes standard Android system tools to efficiently schedule and deliver notifications in the background without draining battery.
  • ইন্টারনেট অ্যাক্সেস — অনলাইন রিসোর্স অ্যাক্সেস এবং নোটিফিকেশনের লেখাগুলো নিরাপদে সিঙ্ক করতে।
  • নেটওয়ার্ক স্টেট অ্যাক্সেস — অনলাইনে কোনো রিকোয়েস্ট পাঠানোর আগে কানেক্টিভিটি চেক করতে।
  • নোটিফিকেশন পারমিশন (POST_NOTIFICATIONS) — অ্যান্ড্রয়েড ১৩ বা তার পরের ভার্সনে পড়াশোনার রিমাইন্ডার এবং অ্যাপের আপডেট জানাতে ব্যবহৃত হয়।
  • ফোরগ্রাউন্ড সার্ভিস/ওয়ার্ক ম্যানেজার — আপনার ডিভাইসের ব্যাটারি সাশ্রয় করে ব্যাকগ্রাউন্ডে নোটিফিকেশনের সময় চেক করার জন্য সিস্টেমের স্ট্যান্ডার্ড টুল ব্যবহার করা হয়।

Remote Sync & Updates

রিমোট সিঙ্ক এবং আপডেট

During startup or usage, the app securely checks GitHub Pages for lightweight files (update.json and notification.json). This enables us to notify you about new features, bug fixes, security patches, and study reminders. No user data or sensitive files are ever accessed, uploaded, or sent to GitHub.

অ্যাপটি চালুর সময় এবং ব্যবহারের মাঝে মাঝে GitHub Pages থেকে update.json এবং notification.json ফাইল চেক করে। এর মাধ্যমে আমরা অ্যাপের নতুন ফিচার, বাগ ফিক্স এবং নোটিফিকেশন সম্পর্কে তথ্য প্রদান করি। গিটহাব থেকে এই ডাটা ফেচ করার সময় আপনার কোনো ব্যক্তিগত বা সেনসিটিভ ডাটা অ্যাক্সেস বা আপলোড করা হয় না।

Children's Privacy
শিশুদের গোপনীয়তা
Educational app privacy approach
শিক্ষামূলক অ্যাপের গোপনীয়তা নীতি

ICT Tool is an educational app. It is designed to minimize personal data handling and does not require account creation.

ICT Tool একটি শিক্ষামূলক অ্যাপ। এটি ব্যক্তিগত ডেটা ব্যবহার কমানোর জন্য ডিজাইন করা হয়েছে এবং এখানে কোনো অ্যাকাউন্ট খোলার প্রয়োজন নেই।

Parents/guardians can remove local app data at any time by clearing app storage or uninstalling the app.

অভিভাবকরা যেকোনো সময় অ্যাপ স্টোরেজ ক্লিয়ার বা আনইনস্টল করে লোকাল ডেটা মুছে ফেলতে পারেন।

Data Security
ডেটা নিরাপত্তা
How local data is protected
লোকাল ডেটা কীভাবে সুরক্ষিত থাকে
  • Data is protected by Android app sandboxing and private app storage.
  • HTTPS is used for network content access and remote JSON syncing.
  • Local preference encryption is used where supported by the device environment.
  • অ্যান্ড্রয়েড অ্যাপ স্যান্ডবক্সিং এবং প্রাইভেট অ্যাপ স্টোরেজ দ্বারা ডেটা সুরক্ষিত।
  • নেটওয়ার্ক কন্টেন্ট এবং রিমোট JSON সিঙ্ক করার জন্য HTTPS ব্যবহার করা হয়।
  • ডিভাইস সাপোর্ট করলে লোকাল প্রেফারেন্সের জন্য এনক্রিপশন ব্যবহার করা হয়।

০৮
Internet Usage
ইন্টারনেট ব্যবহার
When the app uses network access
অ্যাপ কখন ইন্টারনেট ব্যবহার করে

ICT Tool is not fully offline. Internet is used for fetching optional resources and keeping notification messages fresh. Core features run offline.

ICT Tool সম্পূর্ণ অফলাইন নয়। বই বা পিডিএফ আনা এবং নোটিফিকেশনের লেখাগুলো নতুন রাখার জন্য ইন্টারনেট ব্যবহৃত হয়। কোর ফিচারগুলো অফলাইনেই চলে।

  • Update Verification — The app uses the internet to verify new versions and display "What's New" release notes in your preferred language. It only downloads a tiny JSON file, saving your data.
  • Notification Content Sync — The app periodically fetches new notification messages via the internet, consuming very minimal data.
  • Optional content fetch — Triggered when you choose online book/PDF actions.
  • No ad/analytics traffic — No ad or analytics SDK network calls in current build config.
  • আপডেট ভেরিফিকেশন — নতুন ভার্সন চেক করার জন্য এবং 'নতুন কি আছে' (What's New) কন্টেন্টগুলো আপনার পছন্দের ভাষায় দেখানোর জন্য অ্যাপটি ইন্টারনেট ব্যবহার করে। এটি শুধুমাত্র একটি ছোট JSON ফাইল ডাউনলোড করে, যা আপনার ডেটা সাশ্রয় করে।
  • নোটিফিকেশন কন্টেন্ট আপডেট — অ্যাপটি মাঝে মাঝে ইন্টারনেটের মাধ্যমে নোটিফিকেশনের নতুন মেসেজ চেক করে। এটি অত্যন্ত সামান্য ডেটা খরচ করে।
  • ঐচ্ছিক কন্টেন্ট ফেচ — যখন আপনি অনলাইন বই/পিডিএফ দেখার অপশন বেছে নেন।
  • কোনো অ্যাড/অ্যানালিটিক্স ট্রাফিক নেই — বর্তমান কনফিগারেশনে কোনো অ্যাড বা অ্যানালিটিক্স নেটওয়ার্ক কল নেই।

  • Uninstall App — Removes app-local data from the device.
  • In-App Delete Actions — Available for histories, notes, and saved snippets in app flows.
  • ক্লিয়ার অ্যাপ ডেটা — অ্যান্ড্রয়েড সেটিংস -> অ্যাপস -> ICT Tool -> স্টোরেজ -> ক্লিয়ার ডেটা।
  • আনইনস্টল অ্যাপ — ডিভাইস থেকে অ্যাপ-লোকাল ডেটা মুছে ফেলে।
  • অ্যাপের ভেতর ডিলিট অপশন — হিস্ট্রি, নোটস এবং সেভ করা কোড মুছে ফেলার সুবিধা অ্যাপেই রয়েছে।
