01
Information We Collect
What data is stored and where it lives
Privacy-Oriented DesignICT Tool does not require account signup and does not collect personal identity data in-app.

Local Device Storage

The following data may be stored on your device for core app functionality:

  • App Preferences — Theme, language, time format, and feature settings.
  • Calculation History — Previous conversion and arithmetic results.
  • HTML Runner History — HTML snippets entered and saved by you.
  • Study Notes — Notes you create in the Notes section.
  • Reading Progress — Local state related to textbook viewing.
Storage ScopeThis data is stored on your device and can be deleted by clearing app data or uninstalling the app.
02
Information We Do Not Collect
A clear list of data we do not request

Based on the current app configuration, ICT Tool does not request or integrate:

Camera Access Microphone Access Location Access Contacts / SMS Ads SDK Analytics SDK Firebase Analytics In-App Login/Signup

This app does not request sensitive device permissions.

03
Local Storage Usage
How your device stores app data

Data is stored in app-local storage (including SharedPreferences). Some local preferences are protected using encrypted storage where supported.

  • Settings and preferences are saved in app-scoped storage.
  • User-created notes/history remain on your device.
  • No dedicated developer-operated cloud database is used for this data.
Delete AnytimeYou can remove all local data via Android settings (clear app data) or by uninstalling ICT Tool.
04
Permissions Explained
Why each permission is requested

ICT Tool requests only these permissions:

  • Internet Access — Used to access online textbook/PDF resources when you choose those features.
  • Network State Access — Used to check connectivity before online requests.
No Sensitive PermissionsNo camera, microphone, location, contacts, SMS, or storage-management sensitive permissions are requested.

Third-Party Services
External services used by app features

The app may access external content hosts to load textbook PDF resources (like Google Drive) and GitHub Pages to securely check for the latest app version updates.

No Ads/Analytics Libraries in Current Build

AdMob Facebook Ads SDK Firebase Analytics Third-Party Tracking SDKs
06
Children's Privacy
Educational app privacy approach

ICT Tool is an educational app. It is designed to minimize personal data handling and does not require account creation.

Parents/guardians can remove local app data at any time by clearing app storage or uninstalling the app.

07
Data Security
How local data is protected
  • Data is protected by Android app sandboxing and private app storage.
  • HTTPS is used for network content access.
  • Local preference encryption is used where supported by the device environment.
08
Internet Usage
When the app uses network access

ICT Tool is not fully offline. Internet is used for features that fetch textbook/PDF resources. Core calculator and note features can still be used without internet.

  • App Update Checks — The app briefly checks the internet during startup to notify you if a new version is available.
  • Optional content fetch — Triggered when you choose online book/PDF actions.
  • No ad/analytics traffic — No ad or analytics SDK network calls in current build config.
09
Data Deletion
You are always in control
  • Clear App Data — Android Settings -> Apps -> ICT Tool -> Storage -> Clear Data.
  • Uninstall App — Removes app-local data from the device.
  • In-App Delete Actions — Available for histories, notes, and saved snippets in app flows.
